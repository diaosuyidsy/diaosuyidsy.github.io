ALPACA VILLAGE

  • Mobile Action Game
  • Role: Game Designer, Level Designer, QA
  • Date: Summer 2013

Alpaca Village is a 2D Action Game where player scrubs enemy pigs to defend the Village of Alpacas. Interaction is based on the imitation of using a Eraser.

The game is built based on expressing Happiness. Aesthetic is set to be cartoonish and scratchy with all the evil pigs, funny alpacas and scratch book background. Levels are built based on seasons and times in day. All the effect of interactions will remind players of erasing drawings from a draw book.

Realistic sound effect such as erasing, rain drop, wind, bomb etc. are also added to the game based on levels. Unfortunately the game is no longer on App Store because my partner and I no longer own an active apple developer account.
